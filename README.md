# Expense Tracker (Python CLI)

A modular command-line based expense tracking system designed to manage, analyze, and visualize personal expenses.

## Features
- Add, view, and manage expenses
- Category-wise and monthly expense analysis
- Data visualization using Matplotlib
- CSV-based persistent storage
- Payment mode tracking and analysis

## Tech Stack
- Python
- Pandas
- Matplotlib

## Architecture
The system follows a modular design:
- main.py → application control flow
- expense_manager.py → input handling and validation
- storage.py → data storage and retrieval
- analytics.py → reporting and visualization

## Key Highlights
- Implemented data validation and structured storage
- Used Pandas for aggregation and time-based analysis
- Designed modular components simulating scalable system structure
